# Echo Companion Lottie Assets

**Current state:** placeholder — a simple pulsing orange ellipse. This works end-to-end but is visually bare-bones.

**Upgrade path:** search LottieFiles (https://lottiefiles.com/) for terms: `fire sprite`, `magical flame`, `cute flame mascot`, `spirit creature`. Pick a CC-BY or CC0 asset with ideally 4 states (idle, listening, talking, celebrating). If only idle is available, listening/talking/celebrating are derived in-component via Framer Motion transforms.

**Files:**
- `echo-idle.json` — placeholder (to be replaced with real asset)
- `echo-listening.json`, `echo-talking.json`, `echo-celebrating.json` — not present yet; component derives these via Framer Motion until real assets arrive.

When replacing the placeholder, update this file with the source URL, artist credit, and license text.
